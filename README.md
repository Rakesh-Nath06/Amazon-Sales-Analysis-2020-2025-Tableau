## 🎯 Objectives
- Evaluate overall sales performance and growth trends (2020–2025)
- Identify high-performing categories and regions
- Analyze the impact of B2B sales

## 📊 Key Metrics (KPI)
- Total Sales
- Total Orders
- Average Order Value (AOV)
- Top State by Sales

## 📊 Tools Used
- Tableau
- Excel / CSV

## 📁 Dataset
- 5000 rows, 10 columns
- Includes Order ID, Date, Category, Sales Amount, State, B2B, etc.

## 📈 Visualizations
- Stacked Area Chart → Sales Trend
- Filled Map → State-wise Sales
- Pie Chart → Category Share
- Bar Chart → Category Performance
- Heatmap → Order Status Trend

## 🔍 Key Insights
- Total sales reached ₹63.51M from 5000 orders
- Sales show consistent growth from 2020 to 2025
- Uttar Pradesh is the top-performing state
- Non-B2B sales dominate overall revenue
- B2B sales show potential for growth
